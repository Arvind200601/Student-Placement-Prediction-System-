Frequently Asked Questions

Find quick answers about placement readiness and resume prediction.

What is Placement Readiness?

Placement readiness is a score that reflects a student's academic strength, skills, projects, internships, and resume quality.

How is prediction calculated?

The prediction is derived from trained machine learning models using student attributes and resume-derived signals.

Can I upload PDF resumes?

Yes. The system analyzes PDF resumes for skills, projects, internships, and preparation signals.

Which ML algorithm is used?

Random Forest is the primary model, supported by Linear SVM for comparison and confidence estimation.
